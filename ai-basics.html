LLM 大语言模型

Large Language Model · 基石技术

AI 技术栈的最底层基础。通过海量文本数据训练，具备强大的语言理解和生成能力。能够进行对话、写作、翻译、推理等多种任务。

核心特点：基于 Transformer 架构，拥有数十亿到万亿级参数，通过预测下一个词来生成文本。

02

RAG 检索增强生成

Retrieval-Augmented Generation · 知识增强

解决大模型"幻觉"问题和知识时效性的关键技术。在生成回答前，先从外部知识库中检索相关信息，再结合这些信息生成更准确、更有依据的回答。

工作流程：用户提问 → 向量检索相关文档 → 将文档作为上下文 → LLM 生成回答。

向量数据库 Embedding 知识库 语义搜索 Chunking 召回率
03

MCP 模型上下文协议

Model Context Protocol · 标准化接口

Anthropic 提出的开放协议，用于 AI 应用与外部工具、数据源之间的标准化通信。让 LLM 能够安全、统一地访问外部资源（文件系统、数据库、API 等）。

核心价值：类似 AI 领域的 USB 接口，一次开发，到处使用。解决 AI 应用生态碎片化问题。

Anthropic Tool Use Function Calling Server Client Resources Prompts
04

Skill 技能包

AI 能力封装 · 可复用组件

将特定的任务能力或专业知识打包成可复用的模块。一个 Skill 包含 Prompt 模板、工具配置、最佳实践等，让 AI 快速获得特定领域的专业能力。

类比：就像手机 App，每个 Skill 是一个功能模块。用户可以按需安装、组合使用不同的 Skills。

Prompt Template Tool Configuration Best Practices Claude Code 可复用性 模块化
05

Agent 智能体

Autonomous Agent · 自主决策系统

能够自主感知环境、制定计划、执行任务、反思优化的 AI 系统。不同于简单的问答机器人，Agent 具备目标导向、多步推理、工具调用等高级能力。

核心循环：感知 (Perceive) → 规划 (Plan) → 行动 (Act) → 反思 (Reflect)，持续迭代直到目标达成。

ReAct Chain-of-Thought Multi-step Reasoning Memory Planning Tool Use Self-reflection
06

OpenClaw / 应用框架

Application Framework · 终端产品形态

整合上述所有技术的完整应用平台或框架。将 LLM + RAG + MCP + Skill + Agent 融合在一起，提供开箱即用的 AI 助手体验。

典型特征：长期运行、自主执行多步骤任务、管理多个 Agent 协作、集成多种工具和数据源、支持个性化定制。
