📊 分析结果对比

● 路线一：本体 + 大模型

病害诊断：稻瘟病

置信度：85%

防治方案：

使用75%三环唑可湿性粉剂，20-30g/亩，兑水50kg喷雾。破口期和齐穗期各喷一次。

● 路线二：本体 + 大模型 + 小模型

病害诊断：稻瘟病（叶瘟）

置信度：96.8%

防治方案：

诊断确认：稻瘟病-叶瘟型（由稻瘟病菌Magnaporthe oryzae引起）

针对性药剂：
• 首选：75%三环唑可湿性粉剂 25g/亩
• 备选：40%稻瘟灵乳油 80ml/亩
• 生物制剂：枯草芽孢杆菌 100g/亩

用药时机：上午10点前或下午4点后，避免高温用药
安全间隔期：14天

🔧 节点说明

  • start: 输入节点，接收用户上传的水稻叶片图片
  • tool (小模型): 调用 ResNet50 小模型进行图像分类
  • tool (本体/Skill): 调用水稻病害本体 MD 文档作为 Skill
  • llm: LLM节点，结合小模型结果+本体知识生成方案
  • template: 模板节点，格式化输出诊断结果、置信度、防治方案
  • end: 结束节点，返回完整诊断报告

⚡ 特点

  • 小模型（ResNet50）专门训练，识别精度高（96%+）
  • 本体以 Skill/MD 文档形式通过 Tool 节点调用
  • 大模型生成专业、自然的防治方案
  • 三者协同，各发挥所长
