## 1. 核心概念 (Core Concepts)

### 1.1 病害类型 (Disease Types)

#### 稻瘟病 (Rice Blast)
- **定义**: 由稻瘟病菌(Magnaporthe oryzae)引起的真菌性病害
- **发病时期**: 苗期、分蘖期、穗期均可发病
- **发病条件**:
  - 温度: 25-28°C 最适宜
  - 湿度: 高湿度(>90%)持续6小时以上
  - 光照: 阴雨天多发
- **危害程度**: 高，是水稻三大病害之首
- **经济损失**: 严重时可减产40-50%

#### 白叶枯病 (Bacterial Leaf Blight)
- **定义**: 由黄单胞杆菌(Xanthomonas oryzae)引起的细菌性病害
- **发病时期**: 分蘖期至抽穗期
- **发病条件**:
  - 温度: 26-30°C
  - 湿度: 暴风雨期高发
  - 伤口: 台风、暴雨造成伤口
- **危害程度**: 高

#### 纹枯病 (Sheath Blight)
- **定义**: 由立枯丝核菌(Rhizoctonia solani)引起的真菌性病害
- **发病时期**: 分蘖后期至抽穗期
- **发病条件**:
  - 温度: 28-32°C
  - 湿度: 高湿、田间积水
- **危害程度**: 中高

#### 胡麻斑病 (Brown Spot)
- **定义**: 由稻胡麻斑病菌(Bipolaris oryzae)引起的真菌性病害
- **发病时期**: 全生育期
- **发病条件**:
  - 温度: 25-30°C
  - 土壤: 贫瘠、缺钾
- **危害程度**: 中

## 2. 病症特征 (Disease Symptoms)

### 2.1 稻瘟病症状
| 部位 | 症状描述 | 图片特征 |
|------|---------|---------|
| 叶瘟 | 梭形或椭圆形病斑，中央灰白色，边缘褐色 | 灰白色中心，褐色边缘 |
| 穗颈瘟 | 穗颈变褐色，严重时腐烂 | 穗颈变黑，稻穗枯白 |
| 谷粒瘟 | 椭圆形病斑，灰白色 | 稻粒上有褐色斑点 |
| 节瘟 | 节间变褐色，凹陷 | 茎节黑褐色，腐烂 |

### 2.2 白叶枯病症状
| 部位 | 症状描述 | 图片特征 |
|------|---------|---------|
| 叶缘型 | 从叶尖或叶缘开始，沿叶缘向下扩展 | 波纹状黄白色病斑 |
| 急性型 | 叶片灰绿色，向内卷曲枯萎 | 叶片青枯，卷曲 |
| 凋萎型 | 整株凋萎，心叶卷曲 | 整株枯死 |

## 3. 防治方案 (Control Methods)

### 3.1 稻瘟病防治
```yaml
预防措施:
  - 选用抗病品种
  - 合理施肥，避免偏施氮肥
  - 及时排水晒田
  
化学防治:
  药剂名称: 75%三环唑可湿性粉剂
  用法: 20-30g/亩，兑水50kg喷雾
  时机: 破口期和齐穗期
  
  药剂名称: 40%稻瘟灵乳油
  用法: 80-100ml/亩，兑水50kg喷雾
  时机: 发病初期
  
生物防治:
  药剂名称: 枯草芽孢杆菌
  用法: 100g/亩，兑水50kg喷雾
```

### 3.2 白叶枯病防治
```yaml
预防措施:
  - 选用抗病品种
  - 避免在暴风雨前后打田
  
化学防治:
  药剂名称: 20%噻唑锌悬浮剂
  用法: 100-125ml/亩，兑水50kg喷雾
  时机: 发病初期，连喷2-3次
  
  药剂名称: 50%氯溴异氰尿酸可溶粉剂
  用法: 40-50g/亩，兑水50kg喷雾
```

### 3.3 纹枯病防治
```yaml
预防措施:
  - 浅水灌溉，适时晒田
  - 合理密植
  
化学防治:
  药剂名称: 5%井冈霉素水剂
  用法: 150-200ml/亩，兑水50kg喷雾
  时机: 分蘖末期和孕穗期
```
